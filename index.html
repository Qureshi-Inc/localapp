Frequently Asked Questions

❓ How does AI Dev work?

▼
AI Dev reads your GitHub issues, analyzes the codebase using local LLMs like Qwen and DeepSeek, generates solutions automatically, runs tests to ensure quality, and creates pull requests ready for review—all without human intervention.

❓ Do I need API keys or paid services?

▼
No! AI Dev runs entirely locally using LM Studio. You can use free, open-source models like Qwen-7B/14B or DeepSeek-Coder. Zero API costs, complete data privacy.

❓ Is my code data kept private?

▼
Absolutely. All processing happens on your local machine or private infrastructure. Your codebase, issues, and solutions never leave your environment.

❓ What programming languages are supported?

▼
AI Dev works with any language your LLM can understand. It's particularly effective with JavaScript/TypeScript, Python, Go, Rust, and other popular languages.

❓ How does testing work?

▼
Every change generated by AI Dev is automatically tested. The system runs your existing test suite and ensures code coverage stays above 85% before creating a PR.
